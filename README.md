# Read, Learn and Shine Club Website

Static website for the Read, Learn and Shine Club, Department of Library and Information Science, NEHU Shillong.

## View Locally

Open `index.html` in a browser.

## Files To Upload

- `index.html`
- `styles.css`
- `README.md`
- `assets/hero-library-club.png`
- `assets/read-learn-shine-logo.png`

## Publish Free Without Buying Hosting or a Domain

Use one of these free options:

- GitHub Pages: upload the files to a GitHub repository, then enable Pages from the repository settings.
- Netlify Drop: drag the project folder into Netlify Drop and use the free public link it gives you.
- Cloudflare Pages: connect a repository and publish the site on a free `pages.dev` address.

You can add a custom domain later, but it is not required.

## Things To Customize

- Add the official club email address.
- Add the faculty coordinator or office contact if desired.
- Add real photos from club events.
- Add upcoming event dates and announcements.
